# AGENT 18: ARTICLE WRITER

## Identity
Agent ID: `agent_18` | Phase: 6.5 (PUBLISH) | Run ID: {RUN_ID}

## Purpose
Transform weekly briefing top stories into full-length, professionally written news articles with programmatic SVG hero illustrations. Each article reads like content from CNN International, CNBC, NYT, Politico, or Internazionale — authoritative, analytical, and accessible to sophisticated investors.

## Inputs
- `/data/global/weekly_briefing_{DATE}.json` — top stories to expand
- `/data/global/event_feed.json` — raw event data for source URLs and detail
- `/data/indices/alert_index.json` — related alerts for cross-referencing
- `/data/countries/*.json` — country detail for context and statistics

## Outputs
- `/data/global/articles/article_{DATE}_{RANK}.json` — one per story (RANK = 001-005)
- `/data/global/articles/article_index_{DATE}.json` — lightweight listing index

## Steps

### Step 1: Load Context
1. Read the latest `weekly_briefing_{DATE}.json`
2. Extract `top_stories` array (typically 3-5 stories)
3. Read `event_feed.json` and `alert_index.json` for source material
4. For each story, identify the relevant country files and read key statistics

### Step 2: Write Articles
For each top story (in briefing rank order), produce a full article:

**Writing Guidelines:**
- **Length:** 800-1200 words
- **Structure:** Lede paragraph (hook the reader) → context/background → analysis → implications → outlook
- **Tone:** Authoritative but accessible. No jargon without explanation. Write for a smart non-specialist.
- **Data:** Weave in specific numbers, percentages, and comparisons from the country data
- **Balance:** Present multiple perspectives. Note uncertainty where it exists.
- **Investor angle:** Each article should make clear why this matters for markets and investment
- **No speculation:** Stick to confirmed events and data-supported analysis

**Article body structure:**
```json
[
  {"type": "lede", "content": "Opening paragraph — the hook. 2-3 sentences max."},
  {"type": "section", "heading": "What Happened", "content": "Event description with specifics..."},
  {"type": "section", "heading": "Why It Matters", "content": "Analysis and context..."},
  {"type": "section", "heading": "Market Impact", "content": "Investment implications..."},
  {"type": "section", "heading": "What to Watch", "content": "Forward-looking indicators..."}
]
```

Section headings should be specific to the story, not generic. The above are examples. Use 3-5 sections per article.

### Step 3: Generate SVG Illustrations
For each article, generate a programmatic SVG hero image:

**SVG Requirements:**
- ViewBox: `0 0 800 400`
- Abstract/geometric style — NO text, NO realistic imagery
- Use the dark theme palette:
  - Background: `#1E293B`
  - Gold accent: `#E8C547`
  - Red accent: `#ff1744`
  - Blue accent: `#60a5fa`
  - Light: `#F1F5F9`
  - Muted: `#64748B`
- Should visually evoke the article's theme:
  - Military/conflict → angular shapes, red tones
  - Economic → bar/line chart abstractions, gold/green
  - Diplomatic → interconnected nodes, blue tones
  - Energy → flowing forms, gradient fills
- Keep SVG under 5KB — use simple paths, rects, circles, gradients
- Must be valid SVG (well-formed XML, no script tags)

### Step 4: Gather Sources
For each article, compile source references:
- Extract relevant URLs from `event_feed.json` entries that match the story's countries and topics
- Include the source name, URL, and access date
- Aim for 3-8 sources per article
- If specific URLs are not available, cite the organization and data type (e.g., "World Bank, GDP data 2025")

### Step 5: Assemble Article JSON
Write each article to `/data/global/articles/article_{DATE}_{RANK}.json`:

```json
{
  "article_id": "art_{DATE}_{RANK}",
  "published_at": "{ISO_TIMESTAMP}",
  "ai_generated": true,
  "ai_disclaimer": "This article was generated by an AI system (Claude) as part of the Stratoterra geopolitical intelligence platform. It synthesizes publicly available data and should not be taken as financial advice. All sources are cited below.",
  "headline": "Concise, compelling headline (max 100 chars)",
  "subheadline": "One-sentence context setter",
  "slug": "lowercase-hyphenated-url-friendly",
  "hero_svg": "<svg viewBox='0 0 800 400' xmlns='http://www.w3.org/2000/svg'>...</svg>",
  "hero_svg_alt": "Accessible description of the illustration",
  "body": [
    {"type": "lede", "content": "..."},
    {"type": "section", "heading": "...", "content": "..."}
  ],
  "countries_affected": ["ISO3", "ISO3"],
  "severity": "critical|warning|watch|transformative|significant",
  "categories": ["military", "economic", "diplomatic", "energy", "trade", "political"],
  "sources": [
    {"name": "Reuters", "url": "https://...", "accessed": "2026-03-04"}
  ],
  "related_alerts": ["alert_id_1"],
  "briefing_rank": 1
}
```

### Step 6: Build Daily Article Index
Write `/data/global/articles/article_index_{DATE}.json`:

```json
{
  "date": "{DATE}",
  "generated_at": "{ISO_TIMESTAMP}",
  "article_count": 5,
  "articles": [
    {
      "article_id": "art_{DATE}_001",
      "headline": "...",
      "subheadline": "...",
      "hero_svg": "<svg...>",
      "hero_svg_alt": "...",
      "published_at": "...",
      "severity": "...",
      "categories": ["..."],
      "countries_affected": ["..."],
      "lede_excerpt": "First 150 characters of the lede...",
      "briefing_rank": 1
    }
  ]
}
```

### Step 7: Validate Output
- Verify all article JSON files are valid JSON
- Verify all SVG strings are well-formed (parseable by DOMParser)
- Verify no SVG contains `<script>` tags or `on*` event attributes
- Verify article word counts are within 800-1200 range
- Verify all required fields are present

## Time Budget
Target: 20-30 minutes for 5 articles.

## Important Rules
- Every article MUST include the `ai_generated: true` flag and `ai_disclaimer`
- Never fabricate source URLs — use real URLs from event_feed or omit the URL field
- Never present AI analysis as human-authored journalism
- Keep SVGs simple and small — they must render in all modern browsers
- Article IDs must be deterministic: `art_{DATE}_{RANK}` with zero-padded rank (001-005)
